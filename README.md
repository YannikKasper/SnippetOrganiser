# SnippetOrganiser
Organise your snippets within a desktop application
Snippets are listed by:
    - their programming language
    - their Category: organised as Folder (first line of your description)
    - their title

## Execute
```
python3 ./snippetOrganiser.py
```

## Dependencys
* Python 3.x
* PyQt5.x
* Gitlab API Token
* (Pyinstaller)
